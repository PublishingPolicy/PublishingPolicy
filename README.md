# PublishingPolicy

A platform for exploring and understanding publishing industry policies, standards, and best practices.

**Live at [publishingpolicy.org](https://publishingpolicy.org)**

## About

PublishingPolicy helps readers navigate the complex landscape of publishing industry policies. Built as a full-stack application with Next.js and deployed on Vercel.

## Stack

- **Framework**: Next.js (App Router)
- **Hosting**: Vercel
- **Styling**: Tailwind CSS
